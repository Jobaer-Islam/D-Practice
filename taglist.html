CSS Properties Showcase

Box Shadow

The CSS box-shadow property is used to apply one or more shadows to an element.

Rounded Corners

The border-radius property is used to create rounded corners.

Gradient Background

The background property can be used to apply gradient effects.

Styled Border

The border property defines the width, style, and color of an element's border.

Opacity

The opacity property specifies the transparency level of an element.

Transform

The transform property applies 2D or 3D transformations to an element.

Transition

The transition property enables smooth changes between property values.

Animation

The animation property allows elements to transition from one state to another.

Flexbox

The display:flex property is used to layout items in a flexible way.

Grid Layout

The CSS grid property is used to create a grid-based layout.

Text Shadow

The text-shadow property adds shadow effects to text.

Background Image

The background-image property sets an image as the background.

Letter Spacing

The letter-spacing property controls the spacing between letters.

Word Spacing

The word-spacing property sets the spacing between words.

Line Height

The line-height property specifies the height of a line of text.

Z-Index

The z-index property specifies the stack order of elements.

Visibility

The visibility property specifies whether an element is visible or hidden.

Clip Path

The clip-path property clips an element to a specific shape.

Filter

The filter property applies graphical effects like blur and brightness.

Cursor

The cursor property specifies the mouse cursor type when hovering.
